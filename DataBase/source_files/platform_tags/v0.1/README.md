# 平台标签源文件 v0.1

本目录留存当前平台标签导入的 `v0.1` 源文件，作为
`platform_tag_catalog`、`pls_tag_type_dimension_mappings` 和
`pls_tag_value_dimension_mappings` 的可追溯输入。

当前版本包含：

- `1. 天猫_标签类型_标签_20260201.csv`
- `3. 抖音_标签类型_标签_20260201.csv`
- `4. 京东_标签类型_标签_20260714.csv`
- `5. 天猫_AI标签_服饰需求特征_20260714.csv`

其中 `5. 天猫_AI标签_服饰需求特征_20260714.csv` 是用户补充的
AI 服饰需求特征标签，只保留标签值；原始占比不进入本版本源文件。

抖音实际可提取画像标签范围另见：

- `DataBase/source_files/platform_profile_extracts/douyin/v0.1/`

`2. 小红书_标签类型_标签_20260201.csv` 暂不纳入本版本导入范围。

后续平台调整标签时，请新增版本目录，例如 `v0.2`，不要覆盖本目录。
