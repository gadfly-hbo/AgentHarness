# KnowledgeBase Domain Notes

## Owner

Mimo Code。

## 负责

- `KnowledgeBase/**` 内的原始文档、规范、外部资料、引用、来源 metadata、切片、索引和检索。
- 保持知识项到原始来源的可追溯性。

## 不负责

- 把经验教训直接晋升为 MemoryBase 记忆。
- 决定 OntoBase 业务对象或 DataBase schema。

## 强制验证

- 来源、版本、抓取或更新时间明确。
- 切片不丢失必要上下文，索引可重建。
- 检索结果能返回来源引用和适用范围。

## 跨域出口

KnowledgeBase 只通过已批准检索或引用 contract 向其他域提供知识，不要求消费者读取其内部索引实现。
